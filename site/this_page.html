about this page 

This page is proudly created/written/designed without Javascript and dynamic framworking stuff by Norman Köhring.

The tools I used are staticmatic, a lightweight static content framework, HAML and SASS, which are my favorite markup languages for HTML and CSS. Of course I used the one and only editor Vim and last but not least my keyboard ;)

The background and effect images are a self creation by me in coorperation with my good friend Ceiphren. I used GIMP to create them.

This page uses a subset version of the Biolinum2 font from Linux Libertine Open Fonts Project. If you don’t have this font installed, your browser needs to support CSS font downloads (@font-face), otherwise it will look like a regular boring Verdana or Arial. The subset is created with FontSquirrel.

The sources of this page are available at GitHub.
